  플레이오토 계정에서 아래를 먼저 확인해 주시겠어요?

  1. 플레이오토 설정 메뉴에 "자동 발송" 또는 "스케줄" 관련 기능이 있는지
  2. 현재 팝업의 종류 - 로그인 팝업인지, 공지 팝업인지, 광고 팝업인지

  있습니다.

> 공지, 광고 팝업인데 아이프레임 형태야, 1번 기능은 없어

● 아이프레임 팝업은 Playwright로 처리하기 수월합니다. 바로 구현 가능합니다.

  구현 방향

  로그인 → 아이프레임 팝업 감지 → 닫기 버튼 클릭 → 주문내역 페이지 이동 → 엑셀 다운로드

  아이프레임 팝업 처리 핵심 패턴:
  # 방법 A: 아이프레임 안의 닫기 버튼 클릭
  for frame in page.frames:
      close = frame.locator("button:has-text('닫기'), .btn-close, [class*='close']")
      if close.count() > 0:
          close.first.click()

  # 방법 B: 아이프레임 자체를 감싼 오버레이 닫기 (아이프레임 밖에 닫기 버튼)
  page.locator("a[onclick*='closePop'], .popup-close").click()
