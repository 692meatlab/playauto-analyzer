================================================
플레이오토 자동 수집기 사용 방법
================================================

[필요 조건]
- Windows 10/11
- Chrome 브라우저 설치 (대부분 PC에 설치되어 있음)

[설치 방법]

1. 이 폴더를 원하는 위치에 복사

2. config.json 파일 수정
   - playauto_id: 플레이오토 로그인 이메일
   - playauto_pw: 플레이오토 비밀번호
   - 나머지는 수정하지 않아도 됨

3. 스케줄러 등록 (매일 오전 8시 자동 실행)
   - "스케줄러등록.bat" 파일을 우클릭
   - "관리자 권한으로 실행" 선택

[수동 실행]
- PlayautoCollector.exe를 더블클릭

[로그 확인]
- logs 폴더에 실행 기록이 저장됨

[문제 해결]
- Chrome이 설치되어 있는지 확인
- config.json의 ID/PW가 올바른지 확인
- logs 폴더의 로그 파일 확인
- 스크린샷(png 파일)이 있으면 오류 상황 확인 가능

[주의사항]
- PC가 켜져 있어야 스케줄러가 실행됨
- 절전 모드에서는 실행되지 않을 수 있음
